Origin / 初衷

这个项目先是为一个孩子做的。

It began as a practical way to make a difficult school transition feel less overwhelming.

不是替孩子学习，而是把学习路上的噪音降下来。

这个项目最早不是为了做一个“工具”，而是为了帮一个真实的孩子轻一点地走过转轨期。我的儿子今年要参加 International GCSE 大考；他从公办体系转到国际课程还不到一年，课堂语言几乎一下从全中文切换到全英文。知识点本身可以慢慢学，但新的语言、新的考试方式和临近大考的时间压力叠在一起，很容易让孩子觉得自己被推着走。

我用 Codex 做了一个学习、复习用的 Skill：让它先读官方大纲，再把知识点拆成能理解的结构、例题和检查点。这个项目的初衷很简单：利用人工智能帮助孩子更轻松、更有掌控感地面对学业。

That is why the project insists on official sources, bilingual scaffolding, and reviewable outputs. The technology is only useful if it makes learning calmer, clearer, and more humane.

不是题库搬运器，而是大纲驱动的生成流水线。

参考 huashu-design README 的组织方式：先给一句定位，再给可视化效果，再解释能力边界和核心机制。这里的视觉和内容都服务于一个承诺：每个 syllabus fact 都要能回到公开来源。

01 · Discover

找官方 qualification 页面

通过学科名或代码定位 OxfordAQA 公开页面，并记录蓝色 / 红色 listing 元数据。

02 · Extract

下载并解析 specification

运行时下载 PDF，不把官方 PDF 提交到仓库；抽取 topic、assessment 和页码文本。

03 · Compose

生成学习指南结构

组织知识地图、复习路径、例题框架、练习卡和图解规划，保持中英双语模板标签。

04 · Validate

输出前做质量检查

检查 topic、source snippets、PDF、practice cards、diagram 和 provider listing。

生成物长什么样。

学生看到的是漂亮的学习指南；维护者看到的是可追溯数据和校验报告。

HTML / PDF guide

可打印、可直接交给学生预览，但仍保留 review required 的安全边界。

guide-plan.json

结构化保存 topic guide blocks、practice cards、diagram briefs 和 source points。

validation.json

把 topic、assessment、source snippets、listing metadata 和输出完整性放进质量门槛。

孩子愿意看的手册，不能只有文字。

流程是：先根据大纲生成基础知识点和例题；再由 AI 判断哪些知识点或例题需要图文结合；简单图走 SVG，复杂信息图再让用户选择生图模型。

不是文字模板

被 AI 判断为需要图的 topic，必须有图文学习单元：图/信息图、情境问题、讲解风格、答题步骤和 source check。

Visual worked example / 图形例题

模型由用户选择

只有 AI 判断需要复杂信息图时，才询问用户选择 GPT Image、Qwen Image、SenseNova 或自定义模型。

visual_brief + image provider

讲解模板更生动

同一知识点可以用生活场景、侦探推理、动漫闯关感等方式讲，但默认不复刻具体受保护 IP。

Life Scene / Detective / Anime Quest

中英双语不是随便混排。

模板标签可以双语；官方 syllabus terms 不自动翻译。这样看起来少一点“顺滑”，但更稳。

官方内容保留英文

qualification titles、topic titles、paper titles、syllabus points 和 source snippets 都保留 OxfordAQA 原文。

Atomic structure and the periodic table

模板标签统一双语

导航和结构性标题采用中文在前、英文在后的形式，帮助国内学生和家长快速扫读。

知识地图 / Knowledge Map

中文 topic 需要审核词表

如果未来要显示中文 topic 名，必须来自 reviewed glossary 或学科老师复核。
